# Acme Theme

This is an acme theme for Visual Studio.

## Prerequisites

- Visual Studio 2026 (others might work)
- Visual Studio Color Theme Designer 2022 (VS extension)

## Current State

- Basic colors are set to the typical Acme "yellow"
- Syntax Highlighting should be set to "black" font color, with shades of gray for comments
- Colors within text only for errors and other info, used sparingly
